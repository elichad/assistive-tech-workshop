# Workshop: Explore the assistive technology built into your computer

The workshop materials in this repository were originally developed for a 30-minute mini-workshop at Collaborations Workshop 2026. They can be re-used for self-study or to run other workshops.

It is recommended to allow at least an hour to work through all the material for a given operating system.

The material can be browsed at https://elichad.github.io/assistive-tech-workshop/ .
